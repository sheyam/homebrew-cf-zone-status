# Homebrew Tap for Cloudflare Zone Status

A macOS menu bar app for monitoring your Cloudflare zones with security analytics.

## Installation

\`\`\`bash
brew tap sheyam/cf-zone-status
brew install --cask cf-zone-status
\`\`\`

## Requirements

* macOS 13.0 (Ventura) or later
* Cloudflare API Token

See the [main repository](https://github.com/sheyam/cf-zone-status) for details.
